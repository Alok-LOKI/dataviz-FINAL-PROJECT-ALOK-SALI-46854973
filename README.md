## 📌 Project Overview
This project examines shifts in absolute regional emissions, decoupling pathways (GDP vs CO2 per capita), and energy transition structures globally. It utilizes the robust **Our World in Data (OWID) CO2 & GHG Emissions dataset**. 

Key deliverables include:
- A detailed, exploratory Jupyter Notebook with statically generated Plotly visualizations.
- A dynamic, interactive Streamlit Dashboard allowing users to filter, toggle, and explore global emission trends.
- A synthesized PowerPoint presentation outlining methodologies and core conclusions.

## 🚀 How to Run Locally

If you'd like to run the Streamlit dashboard on your local machine:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Alok-LOKI/dataviz-FINAL-PROJECT-ALOK-SALI-46854973.git
   cd dataviz-FINAL-PROJECT-ALOK-SALI-46854973
   ```

2. **Install dependencies:**
   Make sure you have Python installed, then install the required packages:
   ```bash
   pip install -r requirements.txt
   ```

3. **Run the app:**
   ```bash
   streamlit run app.py
   ```
   The dashboard will automatically open in your default web browser at `http://localhost:8501`.

## 📁 Repository Structure
- `app.py`: The main Streamlit dashboard application script.

## 📊 Core Conclusions
1. **The Decoupling Proof:** Major developed nations have successfully decoupled economic growth from CO2 emissions over the past two decades.
2. **Asia's Central Role:** Global decarbonization heavily depends on supporting the industrial and energy transition in developing Asian economies.
3. **Resource Transition:** Bridging from coal to natural gas alongside massive renewable investments is a visible and essential trend among leading economies.
